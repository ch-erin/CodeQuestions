Flex布局全属性详解

Flexible Box Layout（弹性盒布局）是CSS中用于布局的强大工具，以下是所有Flex属性的详细说明和演示

基础概念

Flex布局由两部分组成：

  • Flex容器（Flex Container）：通过设置display: flex的父元素
  • Flex项目（Flex Item）：容器内的直接子元素

Flex布局有两个轴：

  • 主轴（Main Axis）：项目默认沿主轴排列（默认水平方向）
  • 交叉轴（Cross Axis）：与主轴垂直的轴（默认垂直方向）

Flex容器属性

1. display: flex | inline-flex

定义一个Flex容器，将其直接子元素变为Flex项目。

.container {
  display: flex; /* 块级Flex容器 */
  /* 或 */
  display: inline-flex; /* 行内级Flex容器 */
}

5. justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly

定义项目在主轴上的对齐方式。

.container {
  justify-content: flex-start; /* 默认值：左对齐 */
  /* justify-content: flex-end; 右对齐 */
  /* justify-content: center; 居中 */
  /* justify-content: space-between; 两端对齐，项目间距相等 */
  /* justify-content: space-around; 每个项目两侧间距相等 */
  /* justify-content: space-evenly; 项目间距和边缘间距都相等 */
}

  /* row-gap: 10px; 仅设置行间距 */
  /* column-gap: 20px; 仅设置列间距 */

Flex项目属性

1. order: <integer>

定义项目的排列顺序。数值越小，排列越靠前，默认为0。

.item {
  order: 2; /* 数值可以是正数、负数或0 */
}

6. align-self: auto | stretch | flex-start | flex-end | center | baseline

允许单个项目有与其他项目不同的交叉轴对齐方式，可覆盖align-items属性。默认值为auto（继承父元素的align-items）。

.item {
  align-self: center; /* 单个项目居中对齐 */
  /* 其他值与align-items相同 */
}

总结

Flex容器属性（作用于父元素）：

  • display: flex | inline-flex - 定义Flex容器
  • flex-direction - 定义主轴方向
  • flex-wrap - 定义项目是否换行
  • flex-flow - flex-direction和flex-wrap的简写
  • justify-content - 主轴对齐方式
  • align-items - 交叉轴对齐方式（单行）
  • align-content - 交叉轴对齐方式（多行）
  • gap | row-gap | column-gap - 项目间距

Flex项目属性（作用于子元素）：

  • order - 项目排列顺序
  • flex-grow - 项目放大比例
  • flex-shrink - 项目缩小比例
  • flex-basis - 项目基准大小
  • flex - flex-grow、flex-shrink和flex-basis的简写
  • align-self - 单个项目的交叉轴对齐方式
